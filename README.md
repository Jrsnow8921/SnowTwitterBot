# SnowTwitterBot
Cool twitter bot that tweets random facts 


## Install
    git clone git@github.com:Jrsnow8921/SnowTwitterBot.git
    cd SnowTwitterBot
    edit botconfig.py with your Twitter Api credentials.
    
## Run 
    python main.py
  
## Run on a schedule 
    create a cron task to run the main.py file.
